### 组播地址
>239.255.255.251:1982

### 协议格式
    CMD * HTTP/1.1
    location:地址及端口
    method:命令
    param:参数(JSON字符串)


### 支持的命令

>power_on      打开灯
1. effect  [sudden|smooth] (不必填)
2. duration 延时时间 30-n (不必填)


    CMD * HTTP/1.1
    location:192.168.1.100:12345
    method:power_on
    param:{"effect":"smooth","duration":500}

>power_off     关闭灯
1. effect  [sudden|smooth] (不必填)
2. duration 延时时间 30-n (不必填)


    CMD * HTTP/1.1
    location:192.168.1.100:12345
    method:power_off
    param:{"effect":"smooth","duration":500}
